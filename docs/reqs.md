# Requirement Specification

<!-- AI algorithms: Random, Heuristic-Based, Minimax -->
  
## Req 1 Board
UI should display a board containing 3 x 3 tiles.
  
## Req 2 Interactivity
Player should be able to mark one **empty** tile at a time by pressing on it.
  
## Req 3 AI
AI should mark a random **empty** tile when it's their turn.
  
## Req 4 Game End
Game should end when Player or AI has three marks in a row.
  
## Req 5 Replay
UI should contain a replay button that resets the game state (board & AI).
  
## Req 6 Noughts vs Crosses
Player should be able to choose between noughts or crosses. AI should use the unpicked alternative.
  
## Req 7 Difficulty
Player should be able to choose a harder difficulty with advanced AI.
  
## Req 8 Four in a Row
Player should be able to choose a board containing 4 x 4 tiles.
  
## Req 9 State
Game should display current wins and losses. Data should be persistent on reload.
